Attribute Scope Type
neverRunsOnHolidays Public
Notes: Indicates that the validity does not apply on all public holidays no matter if the specified pattern is marked with one for that day. Trains referencing a validity with this value set will not run on public holidays.<br/>
runsOnAllHolidays Public
Notes: Indicates that the validity applies on all public holidays. This means that no matter what is specified via the bitMask, the validity will apply and a train using this validity is scheduled to run.<br/>
runsOnIndicatedHolidays Public
Notes: Indicates that the validity applies even if a day is a public holiday. <br/>
