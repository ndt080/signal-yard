Attribute Scope Type
callOn Public
Notes: This is used for any auxiliary aspect with the meaning “Pass at reduced speed with clear visibility over the route ahead” because the signal cannot be cleared normally. In most cases such aspect is used with a special call-on route.<br/>
caution Public
Notes: This is used for a distant aspect/slave aspect with the meaning “expect Stop” at next signal.<br/>
closed Public
Notes: This is used for any aspect with the meaning “Stop here”.<br/>
combinedProceed Public
Notes: This is used for any proceed aspect where the master and the slave aspect is combined within one single aspect like this were common in OSShD networks. Of course, this applies only to proceed aspects as with the signal closed no slave aspect is given. For any proceed aspects where a speed restriction is expressed with the master aspect the value 'limitedCombinedProceed' shall be used.<br/>
dark Public
Notes: indicates that the signal is switched dark on purpose<br/>
informative Public
Notes: In contrast to supplementary aspects they are giving pure information without any consequences neither to the main aspect nor the train traffic. A failure of this aspect would not affect the main aspect. An example for an informative aspect is any aspect from a direction indicator. It can be also a speed indication if it is relaxing the speed information of the main aspect.<br/>
limitedCombinedProceed Public
Notes: This is used for any proceed aspect where the master and the slave aspect is combined within one single aspect like this were common in OSShD networks. It shall be used for any proceed aspects where a speed restriction is expressed with the master aspect.<br/>
limitedProceed Public
Notes: This is used for any aspect indicating the allowance to continue running with restricted speed. This is typically used for diverging routes or ones with reduced braking distance. It shall be used only if the aspect itself express already a limitation.<br/>In addition this main aspect might be combined with a speed indicator restricting or relaxing the allowed speed against the main aspect.<br/>
markerLight Public
Notes: This is used for an aspect solely marking the signal position but without imposing any limitation. It applies for example to aspect used at shunting signals to indicate local operation allowance.<br/>
proceed Public
Notes: This is used for any aspect indicating the allowance to continue running without any speed restrictions, i.e. proceed with line speed. However, such aspect can be combined on a signal with a speed indicator restricting the allowed speed against the main aspect.<br/>
repeating Public
Notes: The aspect is just the repeating of an aspect of another signal in rear or in advance. This can be the repetition of a distant, shunting or main aspect.<br/>
restriction Public
Notes: This aspect gives an additional restriction to the main aspect, i.e. it is used for speed indicator aspects. A failure of such aspect will affect the main aspect of the signal. An example would be a speed indicator restricting the main proceed aspect.<br/>
shuntingProceed Public
Notes: This aspect is used for allowance of shunting movements without special limitations.<br/>
slowShunting Public
Notes: This aspect is used for allowance of shunting movements but with limited speed only. This may apply for short shunting routes or shunting routes with occupied track sections.<br/>
supplementary Public
Notes: These are any additional signal aspects which are combined with the main aspect without causing a restriction or giving pure information. Such combination shall be supervised by the interlocking and a failure will affect the main aspect as well. A good example is an additional indicator announcing the change onto the wrong track, i.e. line track normally used in the opposite direction.<br/>
warning Public
Notes: This is used for an announcing aspect/slave aspect with the meaning “expect any kind of proceed” at next signal.<br/>
