Attribute Scope Type
auxilliary Public
Notes: The route type is used for auxiliary purpose as part of a complete route (from signal to signal). Often it marks a route part at the interface between two signalboxes (block interface) or the interface between a station and open line (line interface).<br/>
block Public
Notes: This route type is used only on open line. These routes are often automatically handled by the interlocking.<br/>
callOn Public
Notes: This is a special route type for situations, where the interlocking cannot fully secure the route for a train but safety is established by operator rules.<br/>
nonElectrified Public
Notes: This is the route type for securing the path of the trains in the standard way but allows the running path to be without electrification, i.e. it shall be used for diesel or battery trains only.<br/>
normal Public
Notes: This is the route type for securing the path of the trains in the standard way<br/>
occupied Public
Notes: The route leads to an already occupied TVD section, i.e. the destination tracks must be occupied<br/>
shunting Public
Notes: Such routes are typically used for splitting or merging formations of railway vehicles with a reduced amount of safety as the train movement is performed on low speed under responsibility of the driver. Parts of the running path can be occupied<br/>
siding Public
Notes: This special route type is used for exit routes with destination of a siding on open line <i>(de: Ausweichanschlussstelle)</i>. It is used for automatic release of the key lock associated with the siding.<br/>
tunnel Public
Notes: The route leads to a tunnel section, which has restrictions w.r.t. vehicle acceptance, i.e. the vehicles shall be equipped with emergency brake cancellation<br/>
withoutDependency Public
Notes: The route is used when it leads into a non-supervised area, i.e. only the first part is secured by the interlocking. The safety of the complete route path has to be ensured by operator procedures. It applies for example to "telephone block lines", where there is no technical mean to ensure the safe use of the line between two stations but verbal agreement via telephone is used.<br/>
